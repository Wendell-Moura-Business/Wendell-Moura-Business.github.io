Como Funciona o Leilão

Bem-vindo ao nosso sistema de leilão de miniaturas colecionáveis! Aqui está como funciona:

  1. Leilões por tempo limitado: Cada miniatura está disponível para leilão por um período determinado (geralmente 7 dias).
  2. Faça seu lance: Dê lances nos itens que deseja adquirir. Seu lance deve ser maior que o lance atual.
  3. Acompanhe o leilão: Acompanhe o tempo restante e os lances dos outros participantes.
  4. Finalização: Ao término do leilão, o participante com o maior lance é declarado vencedor.
  5. Código de compra: O vencedor pode gerar um código único que garante o direito de compra.
  6. Resgate: Entre em contato conosco via WhatsApp ou e-mail com seu código para finalizar a compra.

Importante: Cada código é único e intransferível. Uma vez gerado, não pode ser regenerado para o mesmo item.

Qualquer dúvida, entre em contato: contato@leilaominiaturas.com.br
